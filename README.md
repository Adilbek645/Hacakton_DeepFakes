# DeepCheck

DeepCheck — это умный ИИ-инструмент для борьбы с дезинформацией, кликбейтом и дипфейками. Проект состоит из браузерного расширения, которое в реальном времени подсвечивает манипулятивный текст на сайтах, и Telegram-бота для мгновенной проверки подозрительных новостей и изображений.

## 🛠 Стек технологий
* **Бэкенд:** Python, Flask, SQLite, Aiogram (Telegram-бот)
* **Фронтенд:** JavaScript (Chrome Extension Manifest V3), HTML, CSS
* **Нейросети и API:** Google Gemini API (анализ текста и эмоций), Sightengine API (распознавание лиц и дипфейков)

## ⚙️ Как запустить проект

### Требования
* Python 3.9+
* API ключ Google Gemini
* Учетные данные API Sightengine

### 1. Настройка сервера и Telegram-бота
Склонируйте репозиторий и установите зависимости:
```bash
git clone https://github.com/yourusername/deepcheck.git
cd deepcheck
pip install -r requirements.txt
```

Создайте файл `.env` в корневой папке и добавьте ваши ключи:
```env
GEMINI_API_KEY=ваш_ключ_gemini
SIGHTENGINE_API_USER=ваш_пользователь_sightengine
SIGHTENGINE_API_SECRET=ваш_секрет_sightengine
BOT_TOKEN=токен_вашего_телеграм_бота
```

Запустите локальный Flask-сервер:
```bash
python app.py
```

В отдельном окне терминала запустите Telegram-бота:
```bash
python bot.py
```

### 2. Установка браузерного расширения
1. Откройте Google Chrome (или Яндекс.Браузер) и перейдите по адресу `chrome://extensions/` (или `browser://extensions/`).
2. Включите **«Режим разработчика»** (Developer mode) в правом верхнем углу.
3. Нажмите **«Загрузить распакованное расширение»** (Load unpacked) и выберите папку `extension` внутри проекта.
4. Откройте любой новостной сайт (например, Пикабу или Дзен), и расширение автоматически начнет проверку текста!

* **Гибкие настройки фильтра:** Возможность выбирать уровень строгости ИИ-цензуры прямо в интерфейсе расширения.
